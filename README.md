# Burn Scar Segmentation — Complete Repo

4 model approaches fine-tuned on YOLOv8 polygon-annotated face scan images.

---

## Repo Structure

```
burn_segmentation/
├── dataset/
│   ├── train/
│   │   ├── images/          ← your 34 training images (already here)
│   │   ├── labels/          ← YOLOv8 polygon .txt files (already here)
│   │   └── masks/           ← binary PNG masks (auto-generated by setup.sh)
│   └── test/
│       ├── images/          ← 3 test images
│       ├── labels/          ← YOLOv8 labels
│       └── masks/           ← binary PNG masks
├── utils/
│   ├── yolo_to_masks.py     ← converts YOLO labels → PNG masks (run once)
│   ├── dataset.py           ← shared PyTorch Dataset class
│   ├── losses.py            ← DiceFocalLoss
│   └── metrics.py           ← Dice, IoU, Precision, Recall
├── method1_sam2/
│   └── train.py             ← SAM 2 fine-tuning
├── method2_medsam/
│   ├── train.py             ← MedSAM fine-tuning
│   └── predict.py
├── method3_unetpp/
│   ├── train.py             ← UNet++ EfficientNet-B4 (START HERE)
│   └── predict.py
├── method4_segformer/
│   ├── train.py             ← SegFormer-B5
│   └── predict.py
├── checkpoints/             ← saved model weights go here
```

## Step 0 — Environment Setup (Run Once)

```bash
# Create environment
conda create -n burn_seg python=3.10 -y
conda activate burn_seg

# PyTorch + CUDA
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118

# All other dependencies
pip install opencv-python-headless pillow numpy scipy matplotlib tqdm \
            albumentations segmentation-models-pytorch \
            transformers accelerate scikit-learn scikit-image

# Convert YOLO polygon labels → binary PNG masks  (mandatory first step)
python utils/yolo_to_masks.py
```

After this, `dataset/train/masks/` and `dataset/test/masks/` will be populated with binary PNG files.

## Method 3 — UNet++ (Start Here — Fastest)

**No extra downloads needed.**

```bash
# Train
python method3_unetpp/train.py

# Predict on test set
python method3_unetpp/predict.py

# Outputs: outputs/unetpp/*_mask.png and *_overlay.png
# Checkpoint: checkpoints/unetpp/best.pth
```

Expected training time: ~30–60 min on RTX 3090  
Expected Dice: 0.75–0.84

---

## Method 4 — SegFormer-B5

**No extra downloads** — HuggingFace downloads the pretrained weights automatically on first run.

```bash
# Train (downloads nvidia/mit-b5 automatically ~340MB)
python method4_segformer/train.py

# Predict
python method4_segformer/predict.py

# Checkpoint: checkpoints/segformer/  (HuggingFace format)
```

Expected training time: ~1–2 hrs  
Expected IoU: 0.78–0.86

## Method 2 — MedSAM

**Requires one manual download (~375 MB).**

```bash
# Step 1: Install SAM
pip install git+https://github.com/facebookresearch/segment-anything.git

# Step 2: Download MedSAM checkpoint
wget -O checkpoints/medsam/medsam_vit_b.pth \
  https://huggingface.co/bowang-lab/MedSAM/resolve/main/medsam_vit_b.pth

# Step 3: Train
python method2_medsam/train.py

# Step 4: Predict
python method2_medsam/predict.py \
  --ckpt_in checkpoints/medsam/medsam_vit_b.pth \
  --ckpt_ft checkpoints/medsam/best.pth
```

Expected training time: ~1–2 hrs  
Expected Dice: 0.82–0.90

---

## Method 1 — SAM 2

**Requires cloning SAM 2 and downloading checkpoint (~900 MB).**

```bash
# Step 1: Clone SAM 2 into the repo root
git clone https://github.com/facebookresearch/segment-anything-2.git
cd segment-anything-2 && pip install -e . && cd ..

# Step 2: Download SAM 2 Large checkpoint
wget -O checkpoints/sam2/sam2_hiera_large.pt \
  https://dl.fbaipublicfiles.com/segment_anything_2/072824/sam2_hiera_large.pt

# Step 3: Train
python method1_sam2/train.py
```

Expected training time: ~2–3 hrs  
Expected Dice: 0.80–0.88

## Recommended Order

1. **Method 3 (UNet++)** — run first, fastest, gives baseline score
2. **Method 4 (SegFormer)** — second, good semantic understanding
3. **Method 2 (MedSAM)** — third, best for medical skin images
4. **Method 1 (SAM 2)** — last, strongest for disconnected burn patches

## Dataset Notes

- **Format**: YOLOv8 segmentation (polygon points, normalized)
- **Train**: 34 images, 9 with burn annotations, 25 background/negative
- **Test**: 3 images (all with burn annotations)
- **Image size**: 640×640 pixels
- **Classes**: 1 (`burn`)
- **Important**: Some train images have no burn region (empty labels) — these are valid negative examples

## Troubleshooting

| Problem | Fix |
|---|---|
| `CUDA out of memory` | Reduce batch size in train.py (BATCH = 1 or 2) |
| `Dice = 0.0` | Check masks exist: `ls dataset/train/masks/` |
| `Module not found` | Make sure conda env is activated: `conda activate burn_seg` |
| `SAM not found` | Run `pip install git+https://github.com/facebookresearch/segment-anything.git` |
| Very low Dice | Dataset is small (9 annotated train images) — augmentation helps a lot |
